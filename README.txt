Hướng dẫn cài đặt và chạy dự án Sokoban

Bước 1: Cài đặt các thư viện cần thiết
Trước tiên, hãy cài đặt các thư viện cần thiết được liệt kê trong tệp `requirements.txt` bằng lệnh sau:

pip install -r requirements.txt


Bước 2: Chạy trò chơi Sokoban
Sau khi cài đặt xong các thư viện, chạy trò chơi bằng lệnh sau:

python sokoban.py
